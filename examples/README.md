# Legacy Examples

These examples belong to the earlier general-purpose executable specification
direction of YUSPEC.

They are still useful for understanding the current C++ compiler/runtime:

- Entity declarations
- Behaviors and state machines
- Events
- Scenarios
- Interpreter behavior

The primary product direction is now Unity gameplay orchestration. New Unity
samples live under:

```text
unity/Packages/com.yuspec.unity/Samples~
```

See `docs/legacy-examples.md` for details.

# YUSPEC Unity v1.0.1 Public Preview

YUSPEC is a text-based gameplay rule layer for Unity.

## Highlights

- Unity Package Manager compatible package
- Entity/event/action/condition gameplay rule runtime
- C# action binding with `YuspecActionAttribute`
- Strict diagnostics for common gameplay rule mistakes
- Editor debugger for specs, entities, events, actions, diagnostics, states, and scenarios
- Working Door+Chest sample
- GoblinAI state-machine sample
- Quest and BossPhase samples
- DemoDungeon sample
- Scenario test subset
- Hot reload subset
- Unity EditMode tests passing in the validated local environment
- Door+Chest scene validation passing in the validated local environment

## Install via UPM

Add this dependency to your Unity project's `Packages/manifest.json`:

```json
{
  "dependencies": {
    "com.yuspec.unity": "https://github.com/Fovane/yuspec.git?path=/unity/Packages/com.yuspec.unity#v1.0.1"
  }
}
```

## Tested With

- Unity 6000.3.8f1
- Unity EditMode tests: passing in the validated local environment
- Door+Chest scene validation: passed in the validated local environment

## Known Limitations

- Public preview release; not yet battle-tested in large production projects
- State machine support is a working subset
- Scenario test support is a working subset
- Hot reload support is a working subset
- Advanced type inference is not included
- Networking, multiplayer replication, visual node editing, shader authoring, and animation authoring are out of scope

## Recommended First Sample

Start with:

```text
unity/Packages/com.yuspec.unity/Samples~/DoorExample
```

## Upgrade Notes

This is the first public preview release line. Existing users of earlier prototype tags should update their Unity manifest dependency to `#v1.0.1`.

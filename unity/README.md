# Unity Workspace

This folder contains:

```text
Packages/com.yuspec.unity  - the local UPM package
YuspecUnityDev             - the Unity development project
```

Open the dev project with Unity 6000.3.8f1. It references the package through a
local file dependency, so package code changes are compiled directly by Unity.

See [docs/unity-dev-environment.md](../docs/unity-dev-environment.md) for commands and validation steps.

# TopDownDungeon GIF Capture Scene

Open `TopDownDungeon.unity` in the Unity dev project and press Play.

Controls:

- `WASD`: move the player capsule
- `Space`: interact with the nearest entity

Suggested GIF path:

1. Interact with the chest in Room 1 to get `Room2Key`.
2. Open the first red door.
3. Move into Room 2 and fight the Goblin with Space.
4. Open the boss-room door after receiving `BossKey`.
5. Enter Room 3, trigger the boss dialogue, fight the Boss, and show the exit door opening.

The scene loads the `.yuspec` rules from:

```text
unity/Packages/com.yuspec.unity/Samples~/TopDownDungeon
```
